These are designed to be typed into a Python interpreter.

Topics to cover:

* 00 - Output
* 01 - Variables
* 02 - Input
* 03 - Primitive Data Types (`bool`, `int`, `float`, `string`)
* 04 - Type conversion
* 05 - Collections 1 (`list` and `tuple`)
* 06 - Collections 2 (`set` and `dictionary`)
